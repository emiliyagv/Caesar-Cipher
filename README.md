# Usage
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; For compiling the program use: 
```
Javac Main.java
``` 
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;for running :
 ```
  Java Main 
 ```
 
 * Enter a word
 * Enter a key which you want to move the word with
 * see the result
