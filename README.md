# Caesar-Cipher
This is an implementation of Caesar Cipher written in Java.
# Author
Emiliya Gaybaliyeva

# Usage
For compiling the program use: 
  Javac Main.java
 
 for running :
  Java Main
 -Enter a word
 -Enter a key which you want to move the word with
 -see the result
